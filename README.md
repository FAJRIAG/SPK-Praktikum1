# SPK-Praktikum1

Sistem Pendukung Keputusan (SPK) Praktikum 1 - Polindra.
Proyek ini merupakan implementasi pengenalan data dan perumusan masalah keputusan menggunakan Flask.

## Persiapan
1. Pastikan Python sudah terinstal.
2. Buat virtual environment: `python -m venv venv`.
3. Aktifkan venv dan instal depedensi: `pip install -r requirements.txt`.
4. Jalankan aplikasi: `python app.py`.
